# Alex Nguyen — Portfolio

Static portfolio site. Hosted on [GitHub Pages](https://pages.github.com).

## Local

Open `index.html` in a browser or use a simple local server (e.g. `npx serve .`).

## Deploy (GitHub Pages)

1. Push this repo to GitHub.
2. **Settings** → **Pages** → **Source**: Deploy from branch.
3. Branch: `main` (or `master`) → folder: **/ (root)** → Save.
4. Site will be at `https://<username>.github.io/<repo-name>/` (or custom domain if set).

## Custom domain (alexnguyen.online)

In repo **Settings** → **Pages** → **Custom domain**: enter `alexnguyen.online`, then add the DNS records GitHub shows (usually a CNAME or A records at your domain registrar).
